# KitchenMind — Agile User Stories (Sprint 1)

| User Story ID | As a (type of user) | I want to (perform some task) | So that I can (achieve some goal) | Priority | Status |
|---------------|----------------------|-------------------------------|-----------------------------------|----------|--------|
| US1 | User | register an account on KitchenMind | securely access my personal kitchen inventory | High | Completed |
| US2 | User | log in to the application | view and manage my own inventory data | High | Completed |
